# FurBLE - Furby Connect Web Bluetooth Demo

Use Web Bluetooth to control your Furby Connect. Tested with Chrome for Android.

### Working
- Connect to / disconnect from Furby
- Send basic commands (e.g. actions, set antenna colour)
